# 規格文件：簡單 3D 射擊遊戲

**Branch**: `001-create-simple-3d-shooter-game` | **Date**: 2026-03-13

## 需求摘要

打造一款可在瀏覽器中直接遊玩的簡單 3D 射擊遊戲，無需安裝任何軟體，可部署於 GitHub Pages。

## 使用者故事

1. **開始遊戲**：玩家進入頁面後可立即點擊開始遊戲，游標被鎖定到畫布上。
2. **移動角色**：使用 WASD 鍵控制角色在 3D 場地中前後左右移動。
3. **瞄準射擊**：移動滑鼠來瞄準，按下滑鼠左鍵發射子彈。
4. **擊殺敵人**：場地中有多個敵人向玩家靠近，子彈命中後敵人消失並得分。
5. **受到傷害**：敵人碰觸玩家時玩家扣血，血量歸零時遊戲結束。
6. **查看分數**：畫面 HUD 顯示當前分數、剩餘血量。
7. **遊戲結束**：血量歸零後顯示 Game Over 畫面，可重新開始。
8. **下一波敵人**：擊殺所有敵人後自動生成更多敵人（波次系統）。

## 功能需求

| 編號 | 需求描述 | 優先度 |
|------|----------|--------|
| F01 | 第一人稱視角 3D 場景（地板、天空、牆壁） | Must |
| F02 | WASD 移動 + 滑鼠視角旋轉（Pointer Lock API） | Must |
| F03 | 滑鼠左鍵射擊，子彈飛行與碰撞偵測 | Must |
| F04 | 敵人 AI：自動朝玩家移動 | Must |
| F05 | 血量系統（玩家預設 100 HP，被碰觸扣 10 HP） | Must |
| F06 | 分數系統（擊殺 +10 分） | Must |
| F07 | HUD：血量條、分數顯示 | Must |
| F08 | 遊戲開始/結束/重開畫面 | Must |
| F09 | 波次系統（每波敵人數量遞增） | Should |
| F10 | 音效（射擊音、受傷音） | Could |

## 非功能需求

- **部署**：純靜態前端，可部署於 GitHub Pages
- **效能**：目標 60 FPS，場景保持簡單多邊形
- **相容性**：支援現代瀏覽器（Chrome、Firefox、Edge）
- **無後端**：全部邏輯於前端執行，無伺服器依賴

## 驗收條件

1. 可在本地 `index.html` 直接以瀏覽器開啟並遊玩
2. WASD + 滑鼠控制可正常運作
3. 子彈發射與敵人消滅邏輯正確
4. 血量歸零顯示 Game Over，可重新開始
5. 單元測試覆蓋核心遊戲邏輯（碰撞偵測、血量、分數）
